Descargar Bookmarklet

Not everyone uses a browser that has an MP4 Downloader add-on available for it. Because of this, we have been developing a "bookmarklet" version of MP4 Downloader. Basically, this version places a button on your bookmarks bar that you can use to download videos from YouTube (although it isn't as feature-full as the MP4 Downloader browser add-on, and only supports YouTube).
