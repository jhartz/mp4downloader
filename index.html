About

MP4 Downloader is a simple, cross-platform, open-source browser add-on that downloads videos from YouTube, Dailymotion, Facebook Video, and Vimeo as MP4 files. This add-on can currently run in Mozilla Firefox (as an add-on) or in any browser as a bookmarklet.

One of the main reasons to download videos in the MP4 format is that it is iPod-compatible without additional conversion. All videos downloaded using the MP4 Downloader can be easily transferred to an iPod (or almost any other portable device).

Terminology

Here is a list of some of the terminology used throughout MP4 Downloader and its documentation.

  • Supported Sites: YouTube, Dailymotion, Vimeo, and Facebook Video
  • Supported Video: a video hosted on one of the Supported Sites
  • Video Page: a web page on one of the Supported Sites that plays a video
  • Embedded Video: a video from one of the Supported Sites that is embedded inside a different web page that is not actually part of the site the video is from (example: this page has embedded YouTube videos in the sidebar on the right, but that web page (an article on Google Help) is definitely not actually YouTube)
