About

MP4 Downloader is a simple, cross-platform, open-source browser add-on that downloads videos from YouTube, Dailymotion, Facebook Video, and Vimeo as MP4 files. This add-on can currently run in Mozilla Firefox (or as a bookmarklet), with Chrome, Safari, Opera, and Mobile Firefox (Fennec) support in the works.

One of the main reasons to download videos in the MP4 format is that it is iPod-compatible without additional conversion. All videos downloaded using the MP4 Downloader can be easily transferred to an iPod (or almost any other portable device).
