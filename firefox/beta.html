Beta Channel

MP4 Downloader operates a beta channel for testing purposes. If you would like to help the development of MP4 Downloader, you can download the latest beta version, test it, and provide feedback.

After you install the beta version, check out the Beta Testing Points page for some areas that need to be tested. Then, if you have any feedback, don't hesitate to contact us.

Installation

You can download the latest beta version from Mozilla Add-ons (scroll down to the bottom of the page and click "Development Channel").

If you're having problems with AMO, try this: Direct Installation Link (version 1.3.3b3)
