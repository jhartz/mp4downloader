Descargar para Mozilla Firefox

Versión 1.3 (Firefox 3.0+)

Haz clic aquí para descargar versión 1.3.3 (la última versión estable).

MP4 Downloader también está disponible de Mozilla Complementos, pero la versión de Mozilla Complementos es más lento en la recepción de actualizaciones cuando una nueva versión es liberada.

Versión 1.2 (Firefox 1.5 - 3.6)

Esta es la rama más antigua para versiones de Firefox más viejas. Esta rama todavía se mantiene con algunas actualizaciones, pero no se añadirán nuevas características. Para descargar la última versión de esta rama (versión 1.2.15), haz clic aquí.
