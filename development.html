Development

Pre-release Versions

Any MP4 Downloader version that is not yet released (ie. not the version available for download on the home page) is considered a "pre-release" version. There are two types of pre-release versions:

  • The beta channel always hosts the latest beta version of MP4 Downloader. Beta versions are considered stable enough for general testing, but not yet stable enough for a public release.
  • Nightly builds are automatically-generated MP4 Downloader builds based on the very latest source code in the CVS repository (below). These builds have no guarantees of stability - code in a nightly build can be completely untested.

Developer Docs

Documentation used in the development of MP4 Downloader is located on the wiki.

Source Code

MP4 Downloader follows an open-source development model. All MP4 Downloader source code is currently hosted at Mozdev and is licensed under the GNU General Public License version 3. The source code can be accessed through one of the following ways:

  • View the source using the CVS web interface:
    • Firefox version
    • Bookmarklet (single file)
  • Check out the source using anonymous CVS with the following commands (password is guest): 
    cvs -d :pserver:guest@mozdev.org:/cvs login
cvs -z3 -d :pserver:guest@mozdev.org:/cvs co mp4downloader

Website Source Code

The MP4 Downloader website is hosted as an open-source project on GitHub. If you have a suggestion for a change, just fork the repo and send a pull request: https://github.com/jhartz/mp4downloader (website content is stored in "gh-pages" branch)
