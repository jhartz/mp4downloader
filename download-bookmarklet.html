Download Bookmarklet

Not everyone uses a browser that has an MP4 Downloader add-on available for it. Because of this, we have been developing a "bookmarklet" version of the MP4 Downloader. Basically, this version places a button on your bookmarks bar that you can use to download videos from YouTube (although it isn't as feature-full as the MP4 Downloader browser add-on, and only supports YouTube).

This new bookmarklet is still a "beta" version, so there may be some bugs. If you find any problems, please contact us.

NOTE: If you are upgrading from an older version of the bookmarklet, make sure to delete the old bookmarklet first (right-click on it and select "Delete"). Then, you can just drag the new version to your bookmarks bar.
