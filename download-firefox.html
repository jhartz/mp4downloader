Download for Mozilla Firefox

Version 1.3 (Firefox 3.0+)

Click here to download version 1.3.2.1 (the latest stable release) from Mozdev.

MP4 Downloader is also available from Mozilla Add-ons, but the Mozilla Add-ons version is usually much slower at receiving updates when a new version is released.

Version 1.2 (Firefox 1.5 - 3.6)

This is the older branch for old versions of Firefox. This branch is still maintained with some updates, but new features will not be added. To download the latest version in this branch (version 1.2.15), click here.
