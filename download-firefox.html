MP4 Downloader

Download for Mozilla Firefox

Version 1.3 (Firefox 3.0+)

Click here to download version 1.3.2 (the latest stable release) from Mozdev. MP4 Downloader is also available from Mozilla Add-ons, but the Mozilla Add-ons version may be much slower at receiving updates when a new version is released.

NOTE: This new version only supports Firefox 3 and newer (in comparison to version 1.2, which supports Firefox 1.5 to Firefox 3.6). If you are still using a version of Firefox prior to Firefox 3, download version 1.2.

Version 1.2 (Firefox 1.5 - 3.6)

This is the older branch for old versions of Firefox. This branch is still maintained with some updates, but new features will not be added. To download the latest version in this branch (version 1.2.14), click here.
