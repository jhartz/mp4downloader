Nightly Builds

If you would like to try out MP4 Downloader nightly builds based on the latest source code in the CVS repository, you can download the latest build from: http://mp4downloader.dyndns.org:1045/nightly/

  • Since nightly builds are based on the latest CVS source code, they are not guaranteed to be very stable, so if you do find bugs, please contact us (make sure to specify that you are using a nightly build).
  • Nightly builds don't automatically update - you will have to download new nightly builds manually.
